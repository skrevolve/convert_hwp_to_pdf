# convert_hwp_to_pdf
법령정보 hwp 파일 pdf 변환

## 필수 사항
- windows OS
- pywin32 (버전 이슈가 있음. python 3.8 이하 권장)
- 한글 hwp 깔려있어야함 (최신버전 hwp파일의 경우 구버전이 깔려있으면 변환할수 없음)

## 설치 및 실행
cmd 를 관리자 권한 허용 상태로 실행시켜야 합니다.

설치  
```cmd
pip install pywin32
```

실행  
```cmd
C:\Users\user\Desktop\convert_hwp_to_pdf > python convert_pdf.py
```
